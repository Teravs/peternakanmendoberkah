# 🐐 Website Mendho Berkah Farm - Wonosobo

Website landing page responsif dan modern untuk peternakan **Mendho Berkah Farm** di Wonosobo, Jawa Tengah. Dibuat dengan struktur HTML5, CSS3, dan JavaScript statis yang ringan, rapi, serta mudah di-deploy via Vercel atau platform hosting statis lainnya.

---

## 🌟 Fitur Utama

- **Layout Modern Split-Hero**: Tampilan beranda dengan dua kolom (Teks & foto) yang memanjakan mata.
- **Katalog 5 Jenis Kambing**:
  - Kambing Merino
  - Peranakan Etawa (PE)
  - Kambing Tokol / Lokal
  - Kambing Dombos
  - Kambing Jawa Randu
- **Pesan Otomatis WhatsApp**: Setiap tombol "Tanya Stok via WA" sudah dikonfigurasi dengan pesan otomatis spesifik sesuai jenis kambing yang dipilih.
- **Galeri Kandang**: Modul galeri 4 foto aktivitas peternakan dengan efek hover zoom modern.
- **Peta Lokasi Google Maps**: Peta interaktif terintegrasi yang mengarah langsung ke lokasi Peternakan Mendho Berkah di Mojotengah, Wonosobo.
- **Navigasi Responsive Mobile**: Dilengkapi *hamburger menu* yang mulus saat dibuka lewat HP/Ponsel.
