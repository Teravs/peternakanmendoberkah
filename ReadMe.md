# Mendo Berkah Website

Website sederhana untuk peternakan Mendo Berkah. Dibuat menggunakan HTML, CSS, dan JavaScript statis.

## Struktur File
- `index.html` : Struktur halaman utama web
- `style.css` : Tampilan warna tema cokelat terracotta dan layout responsif
- `main.js` : Pengatur tombol menu mobile di HP
- `assets/` : Folder tempat logo (`logo.jpeg`) dan foto produk
